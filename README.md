# Boids Swarming Algorithm Implementation
This is an implementation of the Boids algorithm to mimics the movement of a swarming animals. 
This is made for the purpose of completing RSC THT 2 of Aksantara ITB.

# Prerequisites
0. Python 3.13.1
1. Pygame

# How to run
1. `git clone https://github.com/BrianHadianSTEI23/CaRSC26-13523048-BoidsSwarmingAlgorithm.git`
2. `python main.py`
